# 🚖 OLA Data Analytics Dashboard | Power BI + SQL Project

Developed an interactive **OLA Ride Analytics Dashboard** using **Power BI, SQL, and Excel/CSV datasets** to analyze ride booking trends, customer behavior, cancellations, revenue performance, and driver ratings. This project was built on a dataset containing over **100,000 ride records** for Bengaluru city, covering multiple ride categories such as Bike, Auto, Prime Sedan, Mini, eBike, and SUVs. 

The dashboard was designed with a professional multi-page layout including sections for **Overall Performance, Vehicle Type Analysis, Revenue Insights, Cancellation Analysis, and Ratings Overview**. Key business KPIs such as **Total Bookings, Successful Rides, Cancelled Rides, Revenue, Ride Distance, and Customer Ratings** were visualized using interactive cards, bar charts, pie charts, line graphs, and drill-through filters.

Using **SQL queries and views**, ride data was analyzed to extract valuable insights including:

* Successful bookings
* Average ride distance by vehicle type
* Cancellation trends
* Top customers by booking value
* Payment method analysis
* Incomplete rides and reasons
* Driver and customer rating distributions 

Advanced **DAX measures** were created in Power BI for dynamic calculations such as cancellation counts, booking success percentage, revenue metrics, and rating averages. Data cleaning and transformation were performed using **Power Query**, including handling null values, formatting date/time fields, and optimizing relationships between tables.

The project focuses on solving real-world business problems in the ride-booking industry by helping stakeholders understand:

* Peak booking periods
* High-performing vehicle categories
* Customer cancellation behavior
* Revenue distribution by payment methods
* Service quality based on ratings

This dashboard demonstrates strong skills in:

* Data Cleaning & Transformation
* SQL Query Writing
* Data Visualization
* Dashboard Design
* Business Intelligence
* DAX & Power Query
* Analytical Thinking

The project successfully converts raw ride-booking data into meaningful business insights through a clean, modern, and interactive analytics solution suitable for portfolio and real-world business reporting.
